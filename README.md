# Transfory

Transfory is a data transformation toolkit built with Object-Oriented Programming (OOP).  
It includes modules for:
- Missing value imputation  
- Encoding  
- Scaling  
- Feature generation  
- Pipeline automation  
- Insight reporting  

## Setup
Create and activate your virtual environment:
```bash
python -m venv virtenv
source virtenv/Scripts/activate   # or virtenv\Scripts\activate on Windows
